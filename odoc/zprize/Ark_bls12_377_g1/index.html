Module Ark_bls12_377_g1

Bindings to the g1 modulue in the ark-bls12-377 rust package. This is used as a reference implemenetation to test for correctness of our MSM implementation.

Coordinates here are affine coordinates in the regular space (specifically, not montgomery space). The underlying rust implementation uses montgomery space, this is hidden by this module altogether.
